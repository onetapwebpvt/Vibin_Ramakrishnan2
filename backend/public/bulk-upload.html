Instructions

Excel/CSV Format Required:

  1. Create an Excel (.xlsx) or CSV file
  2. First row must be headers (column names)
  3. Required columns:
    • title - Publication title
    • authors - Authors (comma-separated)
    • venue - Journal/Conference name
    • year - Publication year (number)
    • doi - DOI number
    • publisher - Publisher name
  4. Optional columns: order (number), isVisible (true/false)
